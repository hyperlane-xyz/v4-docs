---
title: 'Add IGP Rule'
api: 'POST http://localhost:8080/igp/rules'
---

Add a new Interchain Gas Payment (IGP) rule.

<Note>
This endpoint is only available when gas enforcers are configured.
</Note>

<ParamField body="domain" type="number" required>
  Chain domain ID (u32)
</ParamField>

<ParamField body="min_gas" type="number" required>
  Minimum gas limit (u64)
</ParamField>

<ParamField body="gas_price" type="string" required>
  Gas price in wei (string to handle large numbers)
</ParamField>

<ResponseField name="success" type="object">
  IGP rule created successfully
</ResponseField>

<RequestExample>
```bash curl
curl -X POST "http://localhost:8080/igp/rules" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": 1,
    "min_gas": 21000,
    "gas_price": "1000000000"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:8080/igp/rules', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    domain: 1,
    min_gas: 21000,
    gas_price: "1000000000"
  })
});
```

```python Python
import requests

response = requests.post('http://localhost:8080/igp/rules', json={
    'domain': 1,
    'min_gas': 21000,
    'gas_price': '1000000000'
})
```
</RequestExample>
