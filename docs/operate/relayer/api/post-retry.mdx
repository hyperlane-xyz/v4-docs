---
title: 'Retry Messages'
api: 'POST http://localhost:8080/retry'
---

Trigger retry of failed messages matching specific patterns.

<ParamField body="message_id" type="string">
  Specific message ID to retry (hex string)
</ParamField>

<ParamField body="domain_id" type="number">
  Retry all failed messages for this domain (u32)
</ParamField>

<Note>
Provide either `message_id` for a specific message or `domain_id` to retry all failed messages for a domain.
</Note>

<ResponseField name="success" type="object">
  Retry triggered successfully
</ResponseField>

<RequestExample>
```bash curl
curl -X POST "http://localhost:8080/retry" \
  -H "Content-Type: application/json" \
  -d '{"message_id": "0x123..."}'
```

```javascript JavaScript
const response = await fetch('http://localhost:8080/retry', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    message_id: "0x123..."
  })
});
```

```python Python
import requests

response = requests.post('http://localhost:8080/retry', json={
    'message_id': '0x123...'
})
```
</RequestExample>
