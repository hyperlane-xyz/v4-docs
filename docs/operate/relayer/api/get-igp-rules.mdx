---
title: 'List IGP Rules'
api: 'GET http://localhost:8080/igp/rules'
---

Get current Interchain Gas Payment (IGP) rules.

<Note>
This endpoint is only available when gas enforcers are configured.
</Note>

<ResponseField name="success" type="IGPRule[]">
  Array of IGP rules
</ResponseField>

<ResponseExample>
```json Response
[
  {
    "id": 1,
    "domain": 1,
    "min_gas": 21000,
    "gas_price": "1000000000"
  }
]
```
</ResponseExample>

<RequestExample>
```bash curl
curl "http://localhost:8080/igp/rules"
```

```javascript JavaScript
const response = await fetch('http://localhost:8080/igp/rules');
const rules = await response.json();
```

```python Python
import requests

response = requests.get('http://localhost:8080/igp/rules')
rules = response.json()
```
</RequestExample>
