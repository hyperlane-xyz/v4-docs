---
title: 'Insert Messages'
api: 'POST http://localhost:8080/messages'
---

Insert new messages into the database.

<ParamField body="origin_domain" type="number" required>
  Origin chain domain ID (u32)
</ParamField>

<ParamField body="destination_domain" type="number" required>
  Destination chain domain ID (u32)
</ParamField>

<ParamField body="sender" type="string" required>
  Sender address (hex string)
</ParamField>

<ParamField body="recipient" type="string" required>
  Recipient address (hex string)
</ParamField>

<ParamField body="body" type="string" required>
  Message body (hex encoded)
</ParamField>

<ResponseField name="success" type="object">
  Message inserted successfully
</ResponseField>

<RequestExample>
```bash curl
curl -X POST "http://localhost:8080/messages" \
  -H "Content-Type: application/json" \
  -d '{
    "origin_domain": 1,
    "destination_domain": 2,
    "sender": "0xabc...",
    "recipient": "0xdef...",
    "body": "0x456..."
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:8080/messages', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',

```

```python Python
import requests

response = requests.post('http://localhost:8080/messages', json={
    'origin_domain': 1,

    'recipient': '0xdef...',
    'body': '0x456...'

```
