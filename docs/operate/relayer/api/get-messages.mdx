---
title: 'Get Messages'
api: 'GET http://localhost:8080/messages'
---

Query messages for a specific domain within a nonce range.

<ParamField query="domain_id" type="number" required>
  Chain domain ID (u32)
</ParamField>

<ParamField query="nonce_start" type="number" required>
  Starting nonce (inclusive) (u32)
</ParamField>

<ParamField query="nonce_end" type="number" required>
  Ending nonce (inclusive) (u32)
</ParamField>

<Note>
Validation: Requires `nonce_end > nonce_start`, returns 400 if invalid
</Note>

<ResponseField name="success" type="HyperlaneMessage[]">
  Array of HyperlaneMessage objects
</ResponseField>

<ResponseExample>
```json Response
[
  {
    "id": "0x123...",
    "origin_domain": 1,
    "destination_domain": 2, 
    "nonce": 150,
    "sender": "0xabc...",
    "recipient": "0xdef...",
    "body": "0x456..."
  }
]
```
</ResponseExample>

<RequestExample>
```bash curl
curl "http://localhost:8080/messages?domain_id=1&nonce_start=100&nonce_end=200"
```

```javascript JavaScript
const response = await fetch('http://localhost:8080/messages?domain_id=1&nonce_start=100&nonce_end=200');
const messages = await response.json();
```

```python Python
import requests

response = requests.get('http://localhost:8080/messages', params={
    'domain_id': 1,
    'nonce_start': 100,
    'nonce_end': 200
})
messages = response.json()
```
</RequestExample>
