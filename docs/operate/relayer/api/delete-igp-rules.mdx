---
title: 'Remove IGP Rules'
api: 'DELETE http://localhost:8080/igp/rules'
---

Remove existing Interchain Gas Payment (IGP) rules.

<Note>
This endpoint is only available when gas enforcers are configured.
</Note>

<ParamField body="rule_ids" type="number[]" required>
  Array of rule IDs to delete
</ParamField>

<ResponseField name="success" type="object">
  IGP rules removed successfully
</ResponseField>

<RequestExample>
```bash curl
curl -X DELETE "http://localhost:8080/igp/rules" \
  -H "Content-Type: application/json" \
  -d '{"rule_ids": [1, 2, 3]}'
```

```javascript JavaScript
const response = await fetch('http://localhost:8080/igp/rules', {
  method: 'DELETE',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    rule_ids: [1, 2, 3]
  })
});
```

```python Python
import requests

response = requests.delete('http://localhost:8080/igp/rules', json={
    'rule_ids': [1, 2, 3]
})
```
</RequestExample>
